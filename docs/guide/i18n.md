# 国际化

采用三层合并机制：内置语言包 → 第三方 Provider → 后端 API。

## 使用

```js
import { useI18nStore } from 'nexus-admin-core'

const i18nStore = useI18nStore()

// 切换语言
await i18nStore.setLocale('en')

// 获取当前语言
console.log(i18nStore.locale) // 'zh-CN' | 'en'

// 获取翻译（自动合并三层数据）
console.log(i18nStore.t('common.save'))
```

## 三层合并顺序

```
内置语言包（zh.js / en.js）
       ↓
 第三方 Provider 注册翻译
       ↓
 后端 API 动态语言包
       ↓
  最终 messages（三层深合并）
```

后加载的会覆盖前一层相同 key 的翻译，实现灵活定制。

## 添加翻译

### 在 Provider 中添加

```js
export default {
  name: 'my-app',
  install({ i18nStore }) {
    i18nStore.addMessages('zh-CN', {
      blog: {
        title: '博客管理',
        create: '新建博客'
      }
    })
  }
}
```

### 在组件中使用

```vue
<template>
  <div>{{ t('blog.title') }}</div>
</template>

<script setup>
import { useI18nStore } from 'nexus-admin-core'
const { t } = useI18nStore()
</script>
