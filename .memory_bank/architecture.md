# 架构设计

## 整体架构概览

nexus-admin 是一个 Laravel 扩展包，采用"前端基座 + 后端服务"的架构模式。

```
┌─────────────────────────────────────────────────────┐
│                   前端 (Vite + Vue 3)                │
│  ┌──────────┐ ┌──────────┐ ┌────────────────────┐  │
│  │ 桌面模式  │ │ 侧边栏   │ │  插件系统          │  │
│  │ (Desktop) │ │ (Sidebar)│ │  (Plugin System)   │  │
│  └────┬─────┘ └────┬─────┘ └────────┬───────────┘  │
│       │            │                │              │
│  ┌────┴────────────┴────────────────┴───────────┐  │
│  │          核心层 (Core Layer)                  │  │
│  │  - 布局引擎 (Layout Engine)                   │  │
│  │  - 菜单系统 (Menu System)                     │  │
│  │  - 多窗口/Tab 管理 (Window/Tab Manager)       │  │
│  │  - 主题系统 (Theme System)                    │  │
│  │  - 多语言引擎 (I18n Engine)                   │  │
│  │  - 权限控制 (Permission Control)              │  │
│  │  - 全局配置 (Global Config)                   │  │
│  └───────────────────────────────────────────────┘  │
│                        │                            │
│  ┌─────────────────────┴────────────────────────┐   │
│  │          API 通信层 (Axios)                   │   │
│  └─────────────────────┬────────────────────────┘   │
└────────────────────────┼────────────────────────────┘
                         │ HTTP API
┌────────────────────────┼────────────────────────────┐
│  Laravel 后端          │                            │
│  ┌─────────────────────┴────────────────────────┐   │
│  │         路由 / 控制器 / 服务                  │   │
│  │  - 菜单接口 (Menu API)                       │   │
│  │  - 配置接口 (Config API)                     │   │
│  │  - 语言包接口 (I18n API)                     │   │
│  │  - 权限标签接口 (Permission API)             │   │
│  │  - 模拟接口 (Mock API)                       │   │
│  └──────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
```

## 核心设计原则

1. **基座模式**: 不包含业务逻辑，只提供框架和扩展点
2. **插件化**: 所有功能通过插件机制扩展
3. **统一接口**: 桌面模式和侧边栏模式共享同一套后端接口
4. **配置驱动**: 界面行为由配置控制，支持全局和用户级配置
5. **权限统一**: 基于标签(Tag)的细粒度权限控制，覆盖所有可见元素
